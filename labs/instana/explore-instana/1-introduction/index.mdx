# Introduction

In the third lab of this Instana Jam-in-a-box series you will be exploring the following Instana features:

- Kubernetes and Infrastructure Monitoring capabilities
- Application Perspectives for pre-installed demo applications so you can scope your application views
- Website and Mobile App monitoring
- Unbounded Analytics

The lab should be executed in the numbered order that you see on the left side of the
screen. Some of the sections have dependencies on the previous sections.
